▦ Diagram 🧰 Connector Library
Build your bench here: add harnesses → connectors (mark each P = mates another harness, or J = goes to a sensor/actuator) → circuits (colour + circuit + AWG & mm² + signal + pin). For J connectors, fill the sensor/actuator name, description and photo. Everything is saved automatically and shown in the Diagram (drag boxes there; click a connector for the popup). Use Export JSON to reuse this whole setup for another engine/controller. Connector photos can also be dropped into harness_connectors/ — but uploading here embeds them in the JSON so they travel with the file.
