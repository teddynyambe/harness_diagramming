# Brandvoice — Teddy

Reference file for writing in Teddy's voice. Updated as we work together.

## Words and phrases I use
- "truck in the box" — a benchtop rig that recreates a truck's electrical system for testing
- "benchtesting" / "testbench" / "reproduce it on testbench"
- "raw scan" — the original scanned schematic before any cleanup
- "overview schematic" — a high-level, explained view of a drawing
- "reference item" — a labelled component/connector on a schematic
- "order sensors or actuators" — sourcing parts needed to build a rig

## Style notes
- Prefers concise, direct answers; minimal fluff.
- Wants context files read first, clarifying questions up front, and a brief plan before action.
- Never delete files without explicit approval.

## Domain context
- Works with PACCAR (Kenworth / Peterbilt) truck electrical schematics.
- Goal: understand drawings, extract part/connector info into a text database, and benchtest ("truck in the box").
