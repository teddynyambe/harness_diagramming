HARNESS CONNECTOR IMAGES
========================
Drop a photo of a connector here to have it appear permanently in
"Harness Interconnection Map.html" (instead of picking it each time).

Naming: <slot-key>.jpg  (or .png)
The exact slot-key is shown inside each empty image box in the map,
e.g.  aftertreatment__P417.jpg , cecu3__Conn_C.jpg , engine__Bulkhead.jpg

Tips:
- Keep this folder next to the .html file.
- jpg is tried first, then png.
- If you rename a connector's P/J code in the map, the slot-key changes
  to match the new code (spaces/symbols become "_").
